# 🌍 Scraper CLI Tool

## 📌 Description
Outil en ligne de commande permettant de récupérer des informations sur les pays via web scraping.

## ⚙️ Fonctionnalités
- Scraping web (BeautifulSoup)
- Mise en cache HTML
- Recherche rapide (algorithme dichotomique)
- Interface CLI interactive
- Export en exécutable (.exe)

## 🚀 Utilisation
1. Télécharger `scraper.exe`
2. Lancer le programme
3. Suivre le menu

## 🛠️ Technologies
- Python
- Requests
- BeautifulSoup
- Colorama
